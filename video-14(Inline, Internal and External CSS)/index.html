Three ways to apply CSS

  1. Inline CSS:
    2. Applied directly to an HTML element using the style attribute. Not recommended for large projects or good to avoid.
  2. Internal CSS:
    3. Defined within the <head> section of the HTML document. using the <style> tag. Useful for small projects or when you want to apply styles to a single page.
  3. External CSS:
    4. Defined in a separate .css file and linked to the HTML document using the <link> tag. Recommended for large projects or when you want to apply styles to multiple pages or production rate websites.
